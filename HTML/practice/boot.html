Introduction

Bootstrap is a popular front-end framework. It allows you to quickly design responsive websites using ready-made CSS and JS classes.

Setup

Add these links inside the <head> tag of your HTML:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
